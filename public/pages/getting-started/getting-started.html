Getting started with node.js

The first thing you have to do when starting a new project is to create a package.json file. This file contains information about your project. It is used by npm to keep track of the packages you install and their version numbers.

package.json

{ "name": "my-website", "version": "1.0.0", "dependencies": { "express": "^4.17.1" } }

When you have created the package.json file, you can install the packages you need. In this example we will install the express package. To install a package, you use the npm install command in the console.
