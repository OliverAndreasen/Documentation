Endpoints

HTTP Method Endpoint
GET /songs
GET /songs/<id>
POST /songs
PUT /songs/<id>
PATCH /songs/<id>
DELETE /songs/<id>

Endpoints explained

  • GET /songs: Retrieve a list of songs.
  • GET /songs/<id>: Retrieve a specific song.
  • POST /songs: Add a new song.
  • PUT /songs/<id>: Update an existing song.
  • PATCH /songs/<id>: "patch" an existing song.
  • DELETE /songs/<id>: Delete an existing song.
