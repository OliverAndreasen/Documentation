Serving Files

Different ways to serve files

There are many ways to serve files in node.js.

app.use(express.static(__dirname + '/public'))

This way is to serve the static files to the client(HTML,CSS,JS) these are files can be accessed by the user and should not include any sensitive information.

app.get('/', (req, res) => { res.sendFile(__dirname + '/public/index/index.html') })

This code send a file to the client when a user makes an http request on an endpoint

an example for this endpoint is "http://localhost:8080/"

fs.readFileSync(pagePath).toString()

Here we use the file system module in node.js that helps working with file system on the computer

We use the readFileSync method that reads the contents of the file synchronously (it waits for the file to be read) before the system contiunes
