Dynamic HTML Generation

Creating dynamic HTML templates with the renderPage function

The renderPage function generates dynamic HTML templates by replacing placeholders with values from the config object. This allows for the creation of customized pages where each file can have a new css file if expanend its content

const navbar = fs.readFileSync("./public/components/navbar/navbar.html").toString() .replace("$TAB_TITLE", config.tabTitle || "Documentation") .replace("$CSS_LINK", config.cssLink || "");

The navbar is dynamically generated by reading the contents of the "navbar.html" file. The "$TAB_TITLE" and "$CSS_LINK" placeholders are replaced with the values provided in the "config" object. If no values are provided, the default values are used.
